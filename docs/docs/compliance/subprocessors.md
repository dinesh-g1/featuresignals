# Sub-processors

_Last updated: April 2026_

FeatureSignals uses the following sub-processors to deliver the service. Customers are notified at least 30 days before any new sub-processor is engaged.

## Current Sub-processors

| Sub-processor | Purpose | Data Processed | Location |
|---------------|---------|----------------|----------|
| Cloud hosting provider | Application hosting, database | All service data | As configured by deployment |
| Email service provider | Transactional emails (OTP, notifications) | Email address, name | Provider's infrastructure |

## On-Premises Deployments

For on-premises deployments, FeatureSignals does not act as a sub-processor. All data processing occurs within the Customer's infrastructure. The Customer is responsible for their own sub-processor management.

## Changes

Subscribe to sub-processor change notifications by emailing: privacy@featuresignals.com
